TradingView ile PhillipCapital Webhook Entegrasyon Rehberi

1. PhillipCapital Hesabı Açın

Şimdi Hesap Aç!

Yatırım hesabınızı web sitemizden veya Phillip Mobile G+ uygulamasından açabilirsiniz.

2. TradingView Hesabı Açın ve Uygun Paketi Satın Alın

TradingView Paketleri

“Expert” paketi önerilir. Paket seçiminden önce doğrulama yapılması tavsiye edilir.

3. Phillip Mobile G+ Uygulamasını İndirin

Google Play veya App Store üzerinden uygulamayı indirin. Sonrasında:

  • “Ayarlar” menüsüne girin.
  • “TradingView Sihirbazı”na dokunun.
  • JSON kodunuzu oluşturun.

4. JSON Kodu Nedir ve Neden Önemlidir?

TradingView ve Phillip Mobile G+ arasında senkronizasyonu sağlar. Alarm mesajı alanına yapıştırılır.

5. JSON Kodu Oluşturulurken Dikkat Edilmesi Gerekenler

  • Kod Başlığı: Açıklayıcı bir isim verin. (Örn: AL_GARAN_142₺_Kapanis)
  • İlgili Hisse: Hisse adı doğru girilmeli.
  • İşlem Tipi: Al/Sat bilgisi.
  • Fiyat: Belirli bir fiyat ya da kapanış fiyatı seçilebilir.

Örnek JSON: 

{
   "buyOrSell": "A",
   "orderType": "LMT",
   "seance": "GUN",
   "symbol": "GARAN",
   "quantity": "10",
   "price": "{{close}}"
}

6. TradingView Girişi ve PhillipCapital ile Bağlantı Kurulumu

  1. TradingView hesabınıza giriş yapın.
  2. “Aracı Kurum” sekmesinden PhillipCapital Türkiye’yi seçin.
  3. “Daha Fazla Öğren” → “İşlem” → “Bağlan” adımlarını izleyin.
  4. Giriş bilgilerinizi girerek bağlantıyı tamamlayın.

7. Alarm Oluşturma

  • Alarm simgesine tıklayın, “+” butonuyla yeni alarm oluşturun.
  • Hisse Seçimi: Uygulamadaki ile aynı olmalı.
  • Koşul: Stratejinize veya hazır şablonlara göre belirleyin.
  • Tetikleme: Örn: “Fiyat 100 TL’yi geçerse” gibi.
  • Mesaj: JSON kodunu eksiksiz girin.
  • Webhook URL: API linkini girin.

Son olarak “Oluştur” butonuna tıklayın.

1. Sağ üst taraftan alarm düğmesine bastığımız da çıkan arayüz aşağıdaki gibidir.

2. Mesaj kısmına MobilG+ TradingView sihirbazından aldığımız JSON bilgisini yapıştırırız.

3. Webhook Url kısmına MobilG+ TradingView sihirbazından aldığımız "API Link" bilgisini yapıştırırız.
